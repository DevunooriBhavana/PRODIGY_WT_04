WEATHER APP

TASK01

I developed a Weather App using HTML, CSS, and JavaScript, which provides real-time weather updates for any location. The app utilizes a weather API to fetch data such as temperature, humidity, wind speed, and weather conditions. The user can input a city name, and the app dynamically updates to display the weather information in a clean and responsive interface. JavaScript handles the API calls and data processing, while HTML and CSS are used for structuring and styling the layout, ensuring the app is user-friendly and visually appealing.

TIC TAC TOE

TASK02

The **Tic-Tac-Toe project** is a classic implementation of the popular two-player game using HTML, CSS, and JavaScript. The interface is designed with a clean and responsive layout using HTML and CSS, ensuring a user-friendly experience across devices. The game logic is powered by JavaScript, enabling players to take turns, detect win conditions, and reset the game board. It also incorporates basic AI for single-player mode and real-time updates for multiplayer mode, making it an interactive and engaging project that demonstrates fundamental web development skills.

STOP WATCH

TASK03

For my recent project, I developed a functional stopwatch app using HTML, CSS, and JavaScript. The app's interface is created with HTML, ensuring a clean layout, while CSS adds visual appeal with modern styling. The core functionality is powered by JavaScript, which handles the start, stop, reset, and lap functionalities of the stopwatch. The app effectively utilizes JavaScript's `setInterval` method to track time in real-time, updating the display every second. This project not only demonstrates my ability to create dynamic, user-friendly web applications but also strengthens my understanding of essential front-end development concepts.
